# Soundtrackr 
Aplicação web cliente consumindo APIs externas para busca e interação com músicas.

### APIs utilizadas
iTunes Search API (músicas)
YouTube Data API (vídeos)
Lyrics API (letras)
Wikipedia API (biografia artistas)

### Tecnologias
Python + Flask
HTML + CSS + JS
SQLite (temporário)
Gunicorn (deploy)

### Executar localmente
```bash
pip install -r requirements.txt
python server.py
